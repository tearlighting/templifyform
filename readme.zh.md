# TemplifyForm

一个简单灵活的动态表单生成器。

支持 **Vue3 + Element Plus** 和 **React + Ant Design**，统一数据结构，动态渲染表单项、选项和校验逻辑。

---

## ✨ 特性

- 📦 支持 Vue3（`useForm4Vue`）和 React（`useForm4Rect`）
- 🔥 统一数据层，适配不同前端框架
- 🎯 动态渲染表单模板（输入框、下拉框等）
- 🛡️ 使用 Zod 进行表单验证
- 🛠️ 易于扩展新的输入类型（如 DatePicker、Checkbox）
- ⚡ 少量配置，快速生成复杂表单

---
